# GafferIQ ⚽👔

GafferIQ is a web-based Football Manager simulation game where you take control of real clubs and players from the top 5 European leagues. 

Manage your squad, sign players, scout for wonderkids, tweak your tactics, and watch the match engine simulate 90 minutes of football based on your strategic decisions!

## Features

- **Real Clubs & Players**: Features squads from the English Premier League, La Liga, Serie A, Bundesliga, and Ligue 1.
- **Deep Tactical Engine**: Set your mentality, press intensity, and defensive lines.
- **Live Match Viewer**: Watch your team play out the 90 minutes in an interactive timeline with Half-Time team talks.
- **Transfer Market**: Scout players and negotiate transfer fees and contracts in real-time.
- **Manager Progression**: Earn achievements, build your reputation, and manage your custom avatar's career stats.

## Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- **Next.js 14** (App Router)
- **Tailwind CSS** (for styling)
- **Lucide React** (icons)
- **Claude AI / Heuristic Engine** (Match Simulation)
